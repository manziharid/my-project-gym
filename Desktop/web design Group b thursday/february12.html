Introduction to CSS

CSS stands for cascading style sheet. it is ascripting languange which deals with improving presentation of our web content.

Advantages of CSS

CSS provides more advantages in comparison with HTML. Below are some of them:

  • It saves time while programming
  • It saves storage spaces
  • It reduces the download size
  • It improves the appearances of web content

How to insert CSS code in html

We normally have only 3 main ways of inserting codes in HTML:

  1. Inline CSS:We directly insert CSS codes usig style key word.
  2. Internal CSS: We only put CSS codes in the Head
  3. External CSS: We create a separate file and we put all CSS codes in this file.then we call this file HTML

Types of Selectors

We usually have 4 types of selectors:

  • Element Selectors

  • class Selectors
